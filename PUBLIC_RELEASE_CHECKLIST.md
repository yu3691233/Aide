# AideLink public release staging

This directory is a clean export prepared for a future public repository.

- FRP is optional; LAN access is the default.
- Users may configure any reachable URL (LAN, VPN, reverse proxy, or their own tunnel).
- Tokens, passwords, device state, logs, screenshots, and local IDE bindings must stay outside Git.
- Before publishing, run the repository secret/path scan and replace all remaining private examples with placeholders.

The original development repository and its Git history are intentionally not copied here.
